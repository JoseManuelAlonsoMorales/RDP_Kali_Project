# Instrucciones para preparar el entorno del cliente RDP en Kali Linux

1. Actualización de paquetes y del sistema:

   sudo apt update && sudo apt upgrade -y

2. Instalación de las herramientas necesarias:

   sudo apt install -y xfreerdp3-x11 hydra

3. Reestablecimiento de las configuraciones actuales de red:

   sudo ip addr flush dev eth0

4. Asignación de una nueva ip:

   sudo ip addr add 192.168.10.2/24 dev eth0

5. Activación de la interfaz de red:

   sudo ip link set eth0 up

6. Conexión al servidor RDP:

   xfreerdp3 /u:kali /p:kali /v:192.168.10.1 /cert:ignore

6. Cerrar sesión RDP:

   pkill xfreerdp3
